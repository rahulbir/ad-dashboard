## Instantanea
#### an AppDevices Dashboard

## Installation
####1.  Make sure you have [npm](https://www.npmjs.org/) installed globally

More details here
https://nodejs.org/en/download/ 

####2. On the command prompt run the following commands

```sh
$ npm install 
```
```sh
$ npm start
```

####3. To build in prod
```sh
$ npm run prod
```
